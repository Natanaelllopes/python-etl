# python-etl

Projeto simples de ETL (Extração, Transformação e Carregamento) desenvolvido através de um desafio da DIO,
com foco no entendimento do fluxo ETL (Extração, Transformação e Carregamento) em Python.

## Contexto

Conforme sugerido no desafio, foi utilizada uma abordagem alternativa com dados locais em CSV, dispensando a API e as mensagens com IA generativa.  
A geração de mensagens foi simulada diretamente no código.

## Fluxo ETL

- **Extract**: leitura dos dados a partir de um arquivo CSV
- **Transform**: geração de mensagens personalizadas para cada usuário
- **Load**: gravação dos dados transformados em um novo arquivo CSV
